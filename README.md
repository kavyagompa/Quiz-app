# Quiz App — Java & Automation Testing

A beginner-friendly quiz web app built using HTML, CSS, and JavaScript.

## About
This project tests knowledge of Java, Selenium, TestNG, and Software Testing concepts.
Built as a frontend project to practice HTML structure, CSS styling, and JavaScript logic.

## Features
- 10 multiple choice questions
- Instant right/wrong feedback with explanation
- Score screen with percentage at the end
- Restart option to retake the quiz
- Fully responsive — works on mobile and desktop

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)

## How to Run
1. Clone the repository
2. Open `index.html` in any browser
3. No installations needed
